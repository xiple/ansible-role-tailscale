Ansible role: Tailscale
=========

An ansible role that installs tailscale on Linux.

Requirements
----------------

None.

Role Variables
----------------

None.

Supported distributions
----------------

This role has been been developed and tested on the following distributions :

- Debian 13
- Fedora 42

Example Playbook
----------------

```yaml
- hosts: all
  roles:
    - xiple.tailscale
```
